
MPFI.jl documentation
=====================

*MPFI.jl* is a Julia wrapper of the `MPFI`_ library for multiple precision interval arithmetic. It builds upon what is already available through `BigFloat` to provide an easy-to-use interval arithmetic library.

**Warning**: To use this library, you need to have `MPFI`_ installed and available in your system. Another package fully API-compatible with *MPFI.jl*, without any binary dependencies is planned.
